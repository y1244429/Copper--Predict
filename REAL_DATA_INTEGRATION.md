# 真实数据接入文档

## 概述

系统已成功接入 **AKShare** 真实数据源，可以获取上海期货交易所（SHFE）的铜期货真实价格数据。

## 数据源

### AKShare
- **开源免费**的金融数据接口库
- **数据来源**: 上海期货交易所、伦敦金属交易所等
- **数据类型**: 期货、现货、宏观数据
- **更新频率**: 每日更新

## 已实现功能

### 1. 真实价格数据获取
- **数据源**: AKShare (ak.futures_zh_daily_sina)
- **合约**: SHFE 铜期货主力合约 (CU0)
- **数据内容**:
  - 开盘价 (open)
  - 最高价 (high)
  - 最低价 (low)
  - 收盘价 (close)
  - 成交量 (volume)
  - 成交额 (turnover)
  - VWAP (成交量加权平均价)
  - 持仓量 (hold)
  - 结算价 (settle)

### 2. 自动数据源切换
- 优先尝试 AKShare 获取真实数据
- 如果获取失败，自动回退到模拟数据
- 保证系统稳定性

### 3. 数据统计
- 波动率计算（20日滚动年化波动率）
- 价格统计（最高、最低、平均）
- 价格涨幅计算

### 方式2: 测试数据获取
```python
from data.data_sources import AKShareDataSource

# 创建数据源
source = AKShareDataSource()

# 获取数据
data = source.fetch_copper_price(
    start_date="2025-01-01",
    end_date="2026-02-28"
)

print(f"获取到 {len(data)} 条数据")
print(f"最新价格: {data['close'].iloc[-1]}")
```

## 注意事项

### 1. AKShare 安装
```bash
pip install akshare
```

### 2. 数据获取失败
如果真实数据获取失败，系统会：
- 显示警告信息
- 自动切换到模拟数据
- 继续正常运行

### 3. 网络要求
- 需要互联网连接
- 可能需要访问国际网络（某些数据源）

### 4. 数据延迟
- 真实数据有1-2天延迟
- 模拟数据无延迟
