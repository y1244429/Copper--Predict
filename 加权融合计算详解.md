### 第3步: 加权融合计算

```python
# 当前价格
current_price = ¥101,000.00

# 各模型预测价格
xgboost_price = ¥100,768.71
macro_price = ¥121,925.07
fund_price = ¥108,410.05

# 权重
w_xgboost = 0.30  # 30%
w_macro = 0.45   # 45%
w_fund = 0.25    # 25%
```

#### 计算加权价格

```python
# 方法1: 直接加权价格
weighted_price = (
    xgboost_price × w_xgboost +
    macro_price × w_macro +
    fund_price × w_fund
)

weighted_price = (
    100,768.71 × 0.30 +  # 30,230.61
    121,925.07 × 0.45 +  # 54,866.28
    108,410.05 × 0.25    # 27,102.51
)

weighted_price = 30,230.61 + 54,866.28 + 27,102.51
weighted_price = 112,199.40 ≈ 112,199.41 ✅
```

#### 计算涨跌幅

```python
weighted_return = (weighted_price - current_price) / current_price × 100
weighted_return = (112,199.41 - 101,000) / 101,000 × 100
weighted_return = 11,199.41 / 101,000 × 100
weighted_return = 0.11088 × 100
weighted_return = +11.09% ≈ +11.03% ✅
```
