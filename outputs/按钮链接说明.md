## 🚀 技术实现

### Flask路由
```python
@app.route('/price_curve')
def price_curve_page():
    """价格曲线可视化页面"""
    try:
        with open('templates/price_curve.html', 'r', encoding='utf-8') as f:
            return f.read()
    except FileNotFoundError:
        return "价格曲线页面未找到", 404
```

### HTML链接
```html
<a href="/price_curve">
    <!-- 按钮内容 -->
</a>
```

### JavaScript交互（可选）
```javascript
// 如果需要更多交互控制
document.getElementById('priceCurveButton').addEventListener('click', function(e) {
    e.preventDefault();
    // 添加额外逻辑
    window.location.href = '/price_curve';
});
```
