## ✨ 价格曲线功能特性

### 📊 交互式图表
1. **多模型价格曲线**
   - XGBoost技术模型（5天预测）
   - 宏观因子模型（30天预测）
   - 基本面模型（30天预测）

2. **动态切换**
   - 点击模型按钮可显示/隐藏对应曲线
   - 实时更新图表显示

3. **交互式操作**
   - 鼠标悬停显示详细价格
   - 图例点击可切换模型
   - 缩放和平移支持

### 📈 可视化内容

1. **当前价格横幅**
   - 大字体显示当前价格
   - 渐变背景，视觉效果出色

2. **统计卡片**
   - 各模型预测价格
   - 涨跌幅和预测周期
   - 综合预测结果

3. **主价格曲线图**
   - 多模型对比展示
   - 当前价格参考线
   - 时间轴清晰标注

4. **置信区间图**
   - 显示预测的不确定性范围
   - 95%置信水平
   - 上下限曲线

5. **模型对比柱状图**
   - 各模型涨跌幅对比
   - 颜色区分
   - 数值标签显示

6. **预测说明面板**
   - 各模型特点说明
   - 预测依据介绍
   - 数据来源说明

## 🎨 技术实现

### 使用技术
- **Chart.js**: 专业的JavaScript图表库
- **Chart.js adapter date-fns**: 日期时间轴支持
- **Chart.js plugin annotation**: 参考线标注

### 图表类型
1. **Line Chart**: 价格曲线（带平滑曲线）
2. **Area Chart**: 置信区间填充
3. **Bar Chart**: 模型对比

### 交互功能
- **Toggle**: 模型显示/隐藏切换
- **Tooltip**: 鼠标悬停显示详情
- **Legend**: 图例点击控制显示
- **Responsive**: 自适应屏幕尺寸

## 🎯 使用指南

### 查看价格曲线
1. 访问 http://localhost:8002
2. 点击"📈 查看价格曲线"按钮
3. 或直接访问 http://localhost:8002/price_curve

### 切换模型显示
- 点击顶部的模型按钮（XGBoost / 宏观模型 / 基本面模型）
- 按钮高亮表示显示
- 按钮变灰表示隐藏

### 查看详细信息
- 鼠标悬停在曲线上查看具体日期和价格
- 点击图例控制显示/隐藏
- 滚动缩放查看更多细节

### 导出数据
虽然不再生成PNG图片，但您仍可：
- 下载预测数据CSV: `outputs/predictions.csv`
- 查看JSON报告: `outputs/prediction_report.json`
- 阅读详细报告: `outputs/PREDICTION_SUMMARY.md`

## 🎓 示例操作

### 场景1: 查看XGBoost预测
1. 访问价格曲线页面
2. 点击"XGBoost"按钮（确保高亮）
3. 点击"宏观模型"和"基本面模型"隐藏
4. 查看XGBoost单独曲线和置信区间

### 场景2: 对比多模型
1. 保持所有模型按钮高亮
2. 在主价格曲线图中查看三条曲线对比
3. 观察不同模型的预测差异
4. 查看底部柱状图对比涨跌幅

### 场景3: 查看长期趋势
1. 点击"宏观模型"和"基本面模型"
2. 隐藏XGBoost（短期）
3. 关注30天内的价格变化
4. 查看两个模型的预测差异
