## 🎨 权重策略分析

### XGBoost 高权重 (90%) 的优势

✅ **快速响应市场变化**
- 基于短期技术指标
- 及时捕捉价格波动
- 适合短期交易

✅ **避免基本面滞后**
- 基本面模型反应慢
- 宏观因子模型周期长
- 技术指标更实时

✅ **减少过度乐观**
- 宏观模型+20.66%过于乐观
- 技术模型-0.28%更谨慎
- 避免预测过高

### 潜在劣势

⚠️ **忽视长期趋势**
- 可能错过大牛市机会
- 基本面利好被忽略
- 宏观因子影响被低估

⚠️ **技术指标局限性**
- 技术指标可能失灵
- 无法预测突发事件
- 缺乏深度分析

## 🚀 使用建议

4. **宏观分析** (美元、PMI等影响大)

## 🔧 如何调整权重

### 方法1: 修改代码

编辑 `run_integrated_prediction.py`:

```python
def get_dynamic_weights(self, market_state: str, enhanced_data: dict) -> dict:
    weights = self.base_weights.copy()
    
    # 修改这里的数值
    weights['xgboost'] = 0.90  # XGBoost权重
    weights['macro'] = 0.05    # 宏观权重
    weights['fundamental'] = 0.05  # 基本面权重
    
    return weights
```

### 方法2: 根据市场状态动态调整

```python
if market_state == 'normal':
    weights['xgboost'] = 0.40
    weights['macro'] = 0.35
    weights['fundamental'] = 0.25
elif market_state == 'risky':
    weights['xgboost'] = 0.90  # 风险期技术主导
    weights['macro'] = 0.05
    weights['fundamental'] = 0.05
```
