### 为什么增强系统和集成系统差异这么大？

```
增强系统: ¥93,193.80 (-7.77%)
集成系统: ¥102,106.41 (+1.05%)
差异: +8.82%
```

**原因**:

1. **计算方式不同**:
   - 增强系统: 对价格直接调整
     ```python
     enhanced_price = weighted_price × risk_factor
     = 102,208.60 × 0.9118
     = 93,193.80
     ```

   - 集成系统: 对收益率调整后再计算价格
     ```python
     integrated_return = weighted_return × risk_factor
     = 1.15% × 0.9118
     = 1.05%

     final_price = current_price × (1 + 1.05%)
     = 101,050.00 × 1.0105
     = 102,106.41
     ```

2. **市场状态调整**:
   - 增强系统: 不应用
   - 集成系统: 风险期不调整 (×1.0)

3. **调整对象不同**:
   - 增强系统: 对融合后的价格调整
   - 集成系统: 对收益率调整，再计算价格
