## 💡 风险调整逻辑

### 判断逻辑

```python
def apply_risk_adjustment(prediction, enhanced_data):
    risk_factor = 1.0  # 默认无调整
    
    # 仅在有风险信号时才调整
    if risk_signals:
        for signal in risk_signals:
            if signal['level'] == 'high':
                # 高风险：大幅调整
                adjustment = 调整因子_high
            elif signal['level'] == 'medium':
                # 中风险：适度调整
                adjustment = 调整因子_medium
            
            risk_factor *= adjustment  # 累乘
    
    adjusted_price = prediction × risk_factor
    return adjusted_price
```

### 累乘原理

```python
# 多个风险信号时，调整因子累乘
risk_factor = 1.0
           × 0.94 (美元偏高)
           × 0.97 (VIX偏高)
           = 0.9118

# 累积调整幅度
总调整 = 1 - 0.9118 = -8.82%
```

## ⚠️ 重要说明

### 1. 调整因子的作用

- **小于 1.0**: 向下调整（降低预测）
- **等于 1.0**: 无调整（中性）
- **大于 1.0**: 向上调整（提高预测）

### 2. 累乘效应

多个风险信号时，调整因子累乘：
- 单一信号：轻微调整
- 多个信号：累积调整

### 3. 风险等级

| 信号 | 调整力度 | 说明 |
|------|---------|------|
| 无信号 | 0% | 无调整 |
| 中等 | 3-6% | 适度调整 |
| 高 | 8-12% | 大幅调整 |

### 4. 置信度判断

- **High**: 无风险信号，预测可靠
- **Medium**: 有风险信号，预测谨慎
- **Low**: 多重风险信号，预测保守
