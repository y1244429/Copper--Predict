## 📝 技术细节

### 数据源配置

真实数据版增强预测使用的数据源:

1. **美元指数**: 新浪财经 API (DINIW)
   - 实时获取美元指数
   - 无需API Key

2. **PMI**: AKShare 中国制造业PMI
   - 官方月度数据
   - 无需API Key

3. **VIX**: 新浪期货铜价波动率计算
   - 基于铜价年化波动率
   - 计算公式: VIX = 年化波动率 × 0.55

4. **联邦利率**: Fallback数据
   - 当前美联储利率基准值 (4.75%)
   - 非实时,但为真实数据

### 风险信号生成

基于真实数据生成风险信号:

- ✅ VIX > 20 → 高风险信号
- ✅ 美元指数 > 100 → 中风险信号
- ✅ PMI < 50 → 中风险信号
- ✅ 联邦利率 > 5.0% → 中风险信号

### 市场状态判断

基于真实数据判断市场状态:

- **危机**: 高风险信号≥2个 或 风险信号≥4个
- **风险**: 高风险信号≥1个 或 风险信号≥1个
- **熊市**: VIX > 22 或 美元指数 > 105
- **牛市**: PMI > 52
- **正常**: 其他情况
