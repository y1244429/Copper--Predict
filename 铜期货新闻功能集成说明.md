# 铜价预测系统新闻功能集成说明

## 功能概述

已成功将Tavily AI集成到铜价预测系统，用于实时搜索两种类型的新闻：
1. **科技和AI新闻** - 人工智能、大模型、机器学习等前沿技术
2. **铜期货新闻** - 铜市场行情、价格走势、供需分析

## 主要修改

### 1. 前端界面修改 (`news_futures.html`)

- **科技和AI新闻** - 显示AI、机器学习、深度学习等技术新闻
- **铜期货新闻** - 显示铜市场相关财经新闻
- **期货行情** - 显示期货品种实时行情

### 2. 后端API修改 (`app.py`)

#### 新增功能

1. **Tavily AI双频道集成**
   - **科技和AI频道**: 搜索人工智能、大模型、机器学习等
   - **铜期货频道**: 搜索铜期货、行情、价格走势等

2. **新闻数据缓存**
   - 独立缓存机制：科技AI新闻和铜期货新闻分别缓存
   - 缓存时长：15分钟
   - 自动刷新机制
   - 避免频繁调用Tavily API

3. **情感分析和影响程度评估**
   - 针对不同类型新闻优化的关键词库
   - 正面关键词：上涨、突破、回暖、强劲、支撑、提振等
   - 负面关键词：下跌、暴跌、承压、低迷、疲软、利空等
   - 影响程度识别：高影响、中等影响

4. **备用机制**
   - Tavily API失败时自动切换到模拟数据
   - 确保服务可用性
   - 每种新闻类型独立的备用方案

## 特性说明

### 1. 双频道新闻系统

- **科技和AI频道**: 人工智能、大模型、机器学习、深度学习
- **铜期货频道**: 铜市场行情、价格走势、供需分析
- **独立缓存**: 两种新闻分别缓存，互不影响

### 2. 智能缓存

- **缓存时间**: 15分钟
- **自动刷新**: 缓存过期后自动获取新数据
- **减少调用**: 避免频繁调用API节省配额
- **独立管理**: 科技AI和铜期货新闻独立缓存

### 3. 错误处理

- **API调用失败**: 自动切换到模拟数据
- **网络错误**: 显示错误提示
- **数据为空**: 使用备用模拟新闻
- **独立降级**: 一种新闻失败不影响另一种
